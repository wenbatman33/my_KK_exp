# KK 搶票助手 🎫

一款 Chrome 瀏覽器擴充功能，協助使用者在熱門票務平台（支援 kktix.com / kktix.cc）快速完成購票流程。

## ✨ 功能特色

- **浮動控制面板** — 直接顯示在購票頁面右上角，無需切換分頁
- **票種優先順序** — 輸入多個關鍵字（逗號分隔），依序優先選擇您最想要的區域
- **自訂購票張數** — 自動點擊加號直到達到目標張數
- **倒數計時搶票** — 設定開售時間，時間一到自動重新整理頁面並立即執行
- **自動勾選同意條款** — 自動勾選「我已閱讀並同意 服務條款 與 隱私權政策」
- **自動點擊下一步** — 條款勾選完成、票數到位後立即點擊「下一步」按鈕
- **設定持久暫存** — 欄位設定自動儲存，重新整理後不會清空
- **可拖曳面板** — 面板可自由移動位置，不擋住頁面內容

## 📦 安裝方式

> 本擴充功能目前未上架 Chrome Web Store，需手動以開發者模式載入。

1. 下載或 clone 此 repo 到本機：

   ```bash
   git clone https://github.com/wenbatman33/myKITX.git
   ```

2. 打開 Chrome，在網址列輸入：

   ```
   chrome://extensions/
   ```

3. 開啟右上角的「**開發人員模式**」開關

4. 點擊「**載入未封裝項目**」，選擇剛才下載的資料夾

5. 出現「KK 搶票助手」卡片即代表安裝成功 ✅

## 🚀 使用方式

1. 前往目標活動的購票頁面（`/events/xxx/registrations/new`）
2. 頁面右上角會自動出現**浮動控制面板**
3. 填入以下設定：
   - **票種優先順序**：例如 `Block A, VIP, 一般` （逗號分隔，左邊優先）
   - **購票張數**：預設 1，最多 8 張
   - **開始搶票時間**：選擇開售時間（不填則立即執行）
4. 點擊「**▶ 開始搶票**」

### 倒數模式流程

```
設定時間 → 倒數計時顯示 → 時間到 → 自動 Reload → 選票 → 勾選同意 → 按下一步
```

### 立即模式流程

```
按下開始搶票 → 選票 → 勾選同意 → 按下一步
```

## ⚙️ 技術細節

- **Manifest V3** Chrome Extension
- 針對 AngularJS 架構的票務網站，使用 `angular.element().scope().$apply()` 操作 Data Binding，確保勾選同意條款後能解鎖下一步按鈕的 `ng-disabled` 狀態
- 設定儲存使用 `chrome.storage.local`（跨 reload 持久化）與 `sessionStorage`（傳遞搶票啟動狀態）
